# SMILES (Simplified Molecular Input Line Entry System)

- 규칙

  1. 원자는 표준 원소기호로 나타낸다.
     - 수소는 H, 산소는 O....
  2. 수소 원자는 가능한 모든 곳에 연결되어 있다고 하고, 표기에는 생략
  3. 이웃한 원자는 바로 인접해서 작성
  4. 2중 결합은 '='으로 작성, 3중 결합은 '#'으로 작성
  5. 가지는 괄호 '()'을 사용
  6. 고리는 고리를 만드는 두 원자에 숫자를 표기하는 것으로 나타냄
     - 방향쪽 고리는 원자를 소문자로

  - 표기가 없은 부분은 C로 작성
